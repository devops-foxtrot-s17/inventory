Inventory REST API Service

GET /inventory/products to see all the products.

GET /inventory/products/{id} retrieves a product from the DB using an ID.

PUT /inventory/products/{id} updates a product using an ID.

DELETE /inventory/products/{id} removes a product from the DB using an ID.

POST /inventory/products creates an new product.

Example Data:

    {
        "location_id": "0",
        "new": "1",
        "open_box": "1",
        "product_id": "1",
        "restock_level": "11",
        "used": "1"
    }
